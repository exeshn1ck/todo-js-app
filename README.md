# todo-js-app

Простое приложение для управления задачами, созданное при помощи JavaScript. Проект релаизован для практики работы с DOM, событиями и localStorage.

Демо: [Открыть проект на GitHub Pages](https://exeshn1ck.github.io/todo-js-app/)

Возможности:
 - Добавление, удаление, отмека задач как выполненных
 - Поиск по названию
 - Подсчет количества задач
 - Сохранение состояния в localStorage
 - Очистка выполненных задач

Стек технологий:
 - HTML5
 - CSS3
 - JavaScript(ES6)
 - LocalStorage
 - GitHub Pages
